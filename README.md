## ast

以 vscode 的 "js 代码片断---bst"插件为例，演示如何开发 & 发布 vscode 插件。

- 打包：`vsce package` （非必须，可以直接发布）
- 发布：`vsce publish v0.0.2`

如果想发布自己的 snippet 插件，可以直接 fork 本项目，修改 snippets 里的 snippets.json，和 package.json 里的 publisher 即可。
